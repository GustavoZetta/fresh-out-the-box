# Fresh out of the box
This is a project to the Bad Ideas Game Jam
<br>
Engine made with some code from learnopengl.com

## Main Idea
You are an innocent player 

## Compiling
### Dependencies
The engine uses four dependencies:
* [GLFW](https://www.glfw.org/)
* [GLAD](https://glad.dav1d.de/)
* [GLM](https://glm.g-truc.net/0.9.8/index.html)
* [STB Image](https://github.com/nothings/stb/blob/master/stb_image.h)
<br>
If there are some kind of error involving those, try to download your machine specific binary

### Compilers
The game should compile just fine with **MSVC 2022**, but it may have problems with other compilers

### Running
When running the final game binary, you should open it directly or from the parent terminal if you're opening through terminal
